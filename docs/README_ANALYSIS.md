## ⚠️ 必讀的 3 件事

### 1️⃣ Prompt 只改 3 個地方
```
❌ 不要改：大部分的句子（那些是提醒 AI 的）
✅ 只改：【這裡一定要改】和【這裡可補充】的地方
```

### 2️⃣ 只給同一次任務的附件
```
❌ 不要：混合不同日期/不同網站的材料
✅ 只給：同一次 collect.ps1 執行的完整材料
```

### 3️⃣ 不要上傳敏感資訊
```
❌ 不要上傳：.env、密碼、token、帳號
✅ 可以上傳：aria-snapshots、screenshots、recordings、metadata.json、logs
```
