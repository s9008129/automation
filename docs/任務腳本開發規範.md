# 任務腳本開發規範

> **適用對象**：開發人員、AI 生成腳本時的參考標準
> **重要**：AI 生成任務腳本時必須嚴格遵守本文件所有規範。

---

## 基本原則

1. **離線優先**：所有功能必須在無網際網路的內網環境運作
2. **零系統依賴**：不假設目標電腦有安裝任何工具（Node.js 等皆不依賴系統 PATH）
3. **ESM 模組**：使用 `import/export`，禁止 `require()`
4. **中文訊息**：面向使用者的提示、錯誤、狀態訊息一律使用正體中文

---

## 腳本放置位置與命名

```
src\{系統代碼}-{任務名稱}.ts
```

範例：
- `src\qiz-批次簽核.ts`
- `src\eip-公文簽核.ts`
- `src\vst-資料匯出.ts`

**禁止**將任務腳本放在專案根目錄。

## 標準腳本結構

```typescript
/**
 * 📋 任務腳本說明
 *
 * 執行方式：
 *   .\run-task.ps1 src\你的腳本.ts
 */

// ---- ESM imports（禁止 require）----
import * as fs from 'fs';
import * as path from 'path';
import { fileURLToPath } from 'url';

// ---- 從 src/lib/ 匯入共用 helper ----
import { loadDotEnv, requireEnv } from './lib/env.js';
import { log, logError, initLogger, printHeader, getTaipeiTaskTimestamp } from './lib/logger.js';
import { launchTaskBrowser, closeTaskBrowser } from './lib/browser.js';

// ---- ESM 的 __dirname 等價（必須） ----
const __filename = fileURLToPath(import.meta.url);
const __dirname = path.dirname(__filename);

// ---- 常數 ----
const TASK_NAME = '任務名稱';
const LOG_DIR = path.join(__dirname, '..', 'logs');

// ---- 主要邏輯 ----
async function runTask(): Promise<void> {
  loadDotEnv();
  // ... 實作
}

// ---- 入口點 ----
(async () => {
  initLogger(LOG_DIR, `${TASK_NAME}-${getTaipeiTaskTimestamp()}`);
  printHeader(`🤖 RPA 任務：${TASK_NAME}`);
  try {
    await runTask();
    log('🎉', `任務「${TASK_NAME}」執行完成`);
    process.exit(0);
  } catch (err) {
    logError(`任務「${TASK_NAME}」執行失敗`, err);
    process.exit(1);
  }
})();
```

## 共用 Helper 模組（src/lib/）

### `src/lib/env.ts` — 環境變數

```typescript
import { loadDotEnv, requireEnv, getEnv } from './lib/env.js';

// 載入 .env（自製，不用 dotenv）
loadDotEnv();

// 取得必要環境變數（不存在時拋出中文錯誤）
const password = requireEnv('PASSWORD', '系統密碼');

// 取得可選環境變數（有預設值）
const url = getEnv('EIP_URL', 'https://default.example.com');
```

### `src/lib/logger.ts` — 日誌

```typescript
import { log, logError, initLogger, printHeader, printSection,
         getTaipeiISO, getTaipeiTaskTimestamp } from './lib/logger.js';

// 初始化日誌檔
initLogger(LOG_DIR, `任務名稱-${getTaipeiTaskTimestamp()}`);

// 一般日誌（台北時區時間戳記）
log('✅', '登入成功');
log('⚠️', '找不到元素，略過', 'WARN');
log('❌', '連線失敗', 'ERROR');

// 錯誤日誌（附堆疊資訊）
logError('執行失敗', error);

// 顯示區塊標題
printHeader('登入流程');
printSection('處理表格');
```

### `src/lib/browser.ts` — 瀏覽器

```typescript
import { launchTaskBrowser, closeTaskBrowser, getNestedFrame,
         waitForNavigation, takeScreenshot } from './lib/browser.js';

// 啟動 Edge（任務執行模式）
const ctx = await launchTaskBrowser({
  channel: 'msedge',  // 使用系統已安裝的 Edge
  headless: false,     // 顯示瀏覽器視窗（必須）
});
const { page } = ctx;

// 結束後關閉（在 finally 中呼叫）
await closeTaskBrowser(ctx);

// 雙層 iframe（QIZ / EIP 系統常見）
const frame = getNestedFrame(page, 'iframe[name="mainFrame"]', 'iframe[name="v1"]');
await frame.getByRole('button', { name: '確認' }).click();

// 等待 CAS 登入跳轉
await waitForNavigation(page, '**/main.jsp**', 30000, '主畫面');
```

## 關鍵規範（AI 生成必看）

### ✅ 正確 vs ❌ 錯誤

| 項目 | ✅ 正確 | ❌ 錯誤 |
|------|---------|---------|
| 模組語法 | `import { x } from '...'` | `const x = require('...')` |
| 執行器 | tsx（由 run-task.ps1 呼叫） | ts-node、直接呼叫 npx |
| .env 讀取 | 自製 `loadDotEnv()` | `import dotenv from 'dotenv'` |
| 瀏覽器 | `channel: 'msedge'` | `channel: 'chrome'` 或省略 |
| headless | `headless: false` | `headless: true` 或省略 |
| 敏感資訊 | `requireEnv('PASSWORD')` | `const pw = 'abc123'` |
| CAS ticket | `waitForURL('**/main.jsp**')` | URL 中帶 `?ticket=ST-12345` |
| 使用者訊息 | `'❌ 找不到元素'`（正體中文） | `'Element not found'`（英文） |
| 多頁處理 | `while (hasNextPage) { ... }` | 只處理第一頁 |
| 腳本位置 | `src\任務名稱.ts` | 放在根目錄 |

## 多頁處理模式

系統有分頁時，**必須**使用迴圈處理所有頁面：

```typescript
let currentPage = 1;
let hasNextPage = true;

while (hasNextPage) {
  log('📄', `處理第 ${currentPage} 頁...`);

  // 處理本頁邏輯
  // ...

  // 檢查下一頁
  const nextBtn = page.getByRole('button', { name: '下一頁' });
  const isDisabled = await nextBtn.isDisabled().catch(() => true);
  const isVisible = await nextBtn.isVisible().catch(() => false);

  if (!isVisible || isDisabled) {
    hasNextPage = false;
  } else {
    await nextBtn.click();
    await page.waitForLoadState('networkidle');
    currentPage++;
  }
}

log('✅', `共處理 ${currentPage} 頁`);
```
