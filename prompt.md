
# Prompt 入口說明

這個 repo 的一般使用者 Prompt 已統一收斂到：

- [`docs/使用指南-Edge.md`](docs/使用指南-Edge.md) 第 6 節：生成腳本 Prompt

這樣做的目的，是避免 `README`、一般使用指南、AI 協作文件、獨立 prompt 檔各維護一份不同版本，讓非技術使用者負擔更大。

## 什麼時候看這份檔案？

- 如果你是一般使用者：請直接看 `docs/使用指南-Edge.md`
- 如果你是 repo 維護者，要做多檔修改、規劃 agent / fleet 任務：請看 [`Fleet_Prompt.md`](Fleet_Prompt.md)
